# How to Share Dashboard with Client (UK)

## Option 1: Send Files Directly (FREE, PRIVATE)

### Step 1: Create Zip File
1. Go to `C:\Users\GEO\Desktop\`
2. Right-click `Dashboard` folder
3. "Send to" → "Compressed (zipped) folder"
4. Rename to: `Chocoberry_Dashboard.zip`

### Step 2: Upload & Share
**Google Drive** (Recommended):
1. Go to drive.google.com
2. Upload `Chocoberry_Dashboard.zip`
3. Right-click → Share → Get link
4. Copy link and send to client

**WeTransfer** (Alternative):
1. Go to wetransfer.com
2. Upload zip file
3. Enter client's email
4. Click "Transfer"
5. Client receives download link

### Step 3: Email Client
```
Subject: Restaurant Dashboard - Ready to View

Hi [Client Name],

Your dashboard is complete and ready to review.

Download link: [paste Google Drive/WeTransfer link]

To run the dashboard:
1. Download and extract the zip file
2. Open HOW_TO_RUN_DASHBOARD.md
3. Follow the simple setup steps
4. Dashboard opens in your browser

Let me know if you have questions!

Best regards
```
